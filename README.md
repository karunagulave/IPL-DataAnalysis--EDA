# IPL-DataAnalysis--EDA
# 🏏 IPL Data Analysis & Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Indian Premier League (IPL) dataset using Python. The goal is to analyze team performances, player statistics, match outcomes, and uncover meaningful insights through data visualization.

The project demonstrates data cleaning, preprocessing, visualization, and statistical analysis using popular Python libraries.

---

## 📂 Dataset

The analysis is based on two datasets:

- `matches.csv` – Match-level information
- `deliveries.csv` – Ball-by-ball delivery data

---

## 🎯 Objectives

The project answers several IPL-related business questions, including:

- Team with the highest number of wins
- Team with the best winning percentage
- Toss impact on match results
- Highest team scores
- Teams with the most 200+ scores
- Teams that conceded the most 200+ totals
- Innings-wise performance comparison
- Lucky venues for different teams
- Top batsmen by runs
- Top wicket takers
- Orange Cap and Purple Cap analysis
- Venue-wise statistics
- Season-wise trends
- Winning by runs vs wickets
- Most Player of the Match awards

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/karunagulave/IPL-DataAnalysis--EDA.git
```

Move into the project folder:

```bash
cd IPL-DataAnalysis--EDA
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open the notebook and run all cells.
